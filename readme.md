🎥 Infinity V Cam

Virtual Camera Controller for Chrome

Infinity V Cam is a powerful browser extension that provides real-time control over your webcam feed. Designed for creators and professionals, it uses a high-performance Canvas engine to transform your video stream with zero latency.

✨ Core Features

Live Transformation: Smoothly scale ($0.2x$ to $3.0x$) and pan your camera feed.

Mirroring: Instant horizontal and vertical flip controls.

Visual Feedback: Dynamic "No Signal" grain effect when the camera is inactive.

Glassmorphic UI: Modern, intuitive interface built for Chrome.

Multi-Device: Seamless switching between multiple connected cameras.

🚀 Quick Installation

Download the project folder to your computer.

Open Chrome and go to chrome://extensions/.

Turn on Developer Mode (top right).

Click Load unpacked and select the extension folder.

Pin the extension to your toolbar for easy access.

🛠️ Tech Stack

Engine: HTML5 Canvas 2D Context.

UI: Modern CSS with Lavender/Indigo glassmorphism.

Processing: Real-time $1080p$ stream manipulation.
